1️⃣  Create directories & place files (follow project structure above)

2️⃣  Create MySQL database:
    $ mysql -u root -p < schema.sql

3️⃣  Update database credentials:
    Edit: python/config.py
    Change: DB_PASSWORD = 'your_mysql_password'

4️⃣  Install Python dependencies:
    $ pip install -r requirements.txt

5️⃣  Load sample data:
    $ cd python
    $ python main.py --load-sample-data

6️⃣  Run complete analysis:
    $ python main.py

    • CSV Files: ../output/
